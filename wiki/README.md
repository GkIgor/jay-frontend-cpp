# Jay Frontend C++ Wiki

Bem-vindo à documentação do repositório **Frontend** do projeto Jay IA.

Este repositório (`jay-frontend-cpp`) é estritamente focado no corpo da IA: **interface visual (UI) e saída de áudio**. Todo o raciocínio, memória, plugins e planejamento (o "cérebro") estão isolados no Core em Go (`jay-ia`).

## Regra de Ouro (Core vs Frontend)

> [!IMPORTANT]
> **A Wiki do Core é a fonte de verdade para arquitetura do sistema e protocolos de comunicação (IPC).**
> Esta wiki aqui foca exclusivamente nos **detalhes de implementação C++**, fluxo de renderização Raylib, gerenciamento da thread de IPC do lado do cliente, e decisões locais de frontend.
